# System Info

The System Info screen shows details about your computer's hardware and operating
system in one place, so you do not need to open separate system dialogs.

## What You Can Do

- View processor details such as model, speed, cores, and logical processors.
- View memory information.
- View operating-system and device identity details.

## Open System Info

1. In the sidebar, select **System Info**.
2. The details populate from the current system snapshot.

## What You Will See

- **Processor** — model, current and base speed, sockets, cores, logical
  processors, and related details.
- **Memory** — installed and in-use memory and related figures.
- **Device and operating system** — device name, manufacturer, model, and OS
  edition, version, and build (availability depends on your platform).

## Recommended Workflow

1. Open System Info when you need to confirm your hardware or OS details.
2. Use it alongside [Performance](/prism/monitoring/performance) to relate
   capacity (what you have) to usage (how hard it is working).

## Limitations

- Device and operating-system identity fields are most complete on Windows. On
  other platforms some fields may be empty.
