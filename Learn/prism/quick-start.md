# Quick Start

This walkthrough takes you through your first session with Prism: launching the
app, understanding the layout, watching live data, and performing a few common
tasks.

## 1. Launch Prism

1. Start **Prism** from your Start menu (Windows) or Applications (macOS).
2. The window opens on the **Dashboard**.
3. Wait a moment for the first data refresh to populate the metrics.

## 2. Learn the Layout

Prism has three main regions:

- **Title bar** (top) — window controls, a search box, and a **Settings** button.
- **Sidebar** (left) — navigation between screens, plus shortcuts to native
  tools.
- **Work area** (center) — the currently selected screen.

You can resize the sidebar by dragging its right edge.

## 3. Watch Live System Data

1. Stay on the **Dashboard** to see summary metrics for CPU, GPU, memory, disk,
   and network.
2. Open [Performance](/prism/monitoring/performance) for dedicated per-resource
   charts with history.

## 4. Find and End a Busy Process

1. In the sidebar, select **Processes**.
2. Sort by a column such as CPU or memory to find heavy processes.
3. Select the process you want to stop.
4. Use the end/terminate action and confirm if prompted.

By default, Prism asks for confirmation before ending a process. You can change
this in [Settings](/prism/settings/general). See
[Processes](/prism/monitoring/processes) for details.

> Ending a process closes it immediately and any unsaved work in that application
> may be lost.

## 5. Reclaim Disk Space

1. Select **Storage Cleanup** in the sidebar.
2. Run a scan to see reclaimable locations and sizes.
3. Review the results, select what you want to remove, and clean.

See [Storage Cleanup](/prism/tools/storage-cleanup) for safety notes.

## 6. Explore the Tools You Need

Depending on your work, continue with:

- [Startup Apps](/prism/tools/startup-apps) — control what launches at startup.
- [Ports](/prism/tools/ports) — find what is listening on a port.
- [Network Devices](/prism/tools/network-devices) — discover local devices.
- [SSH Keys](/prism/tools/ssh-keys) — list and generate keys.
- [Docker](/prism/tools/docker) — manage containers and images.
- [Command Center](/prism/console/command-center) — use the built-in terminal.

## 7. Configure Prism to Your Liking

1. Open **Settings** from the title bar.
2. Set your tray behavior and startup preference.
3. Enable the native tool shortcuts you use most.
